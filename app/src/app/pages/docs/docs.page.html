Learn how to build high-performance, accessible, and beautiful user interfaces with Quartz UI.

Introduction

Quartz UI is a library of headless, unstyled components for Angular. Our philosophy is to provide the logic and accessibility features (WAI-ARIA) of common UI patterns, while leaving the styling completely in your hands.

Installation

npm install @quartz/ui

Quick Start

Import the primitives you need and start building.

import {{ '{' }} OverlayModule {{ '}' }} from '@quartz/ui'

@Component({{ '{' }}
  selector: 'app-component',
  imports: [OverlayModule],
  template: \`
    <button q-overlay-trigger>Open Popover</button>
    <q-overlay>
      <div class="popover-content">Hello World!</div>
    </q-overlay>
  \`
{{ '}' }})
export class MyComponent {{ '{' }} {{ '}' }}
