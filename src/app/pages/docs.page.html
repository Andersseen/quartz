Getting Started

Quartz UI is a collection of unstyled, accessible, and high-performance Angular primitives. Follow these steps to integrate them into your project.

1. Installation

Add the core package to your project using your preferred package manager.

pnpm add @quartz/ui

2. Styles Configuration

Quartz UI is CSS-agnostic. However, we recommend setting up a basic CSS reset and a color palette in your styles.scss.

styles.scss
// Quartz UI doesn't require any specific CSS imports.
// You are free to use Tailwind, CSS Modules, or Vanilla CSS.

:root {{ '{' }}
  --qz-primary: #8b5cf6;
  --qz-primary-glow: rgba(139, 92, 246, 0.4);
{{ '}' }}

3. Basic Usage

Import the standalone primitive you need and start building.

app.component.ts
import {{ '{' }} QzOverlay {{ '}' }} from '@quartz/ui/overlay';

@Component({{ '{' }}
  selector: 'app-root',
  standalone: true,
  imports: [QzOverlay],
  template: \`
    <button qzOverlayTrigger>Open Menu</button>
    <qz-overlay>
      <div class="menu-content">...</div>
    </qz-overlay>
  \`
{{ '}' }})
export class AppComponent {{ '{' }} {{ '}' }}
